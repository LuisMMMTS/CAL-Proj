Para compilar:
Criar um projeto de Clion baseado no ficheiro CMakeLists.txt presente na raiz do diretório.

Atenção, é necessário comentar uma linha/descomentar no ficheiro CMakeLists que está devidamente indicada, dependendo de
 utilizar S.O. baseado em Unix ou windows (em windows é necessária a biblioteca wsoc32) para a
 utilização de graphviewer.


Instruções de utilização:
Iniciar, e aparecerá um menu intuitivo. Aparecerá um mapa da região selecionada para que o utilizador possa escolher os
ids de inicio e de destino que serão introduzidos a seguir no terminal. Atenção: não fechar esta janela, se pretender
fechar esta janela para o programa prosseguir, carregar em enter no terminal e a janela fechar-se-á automaticamente.

 Posteriormente é calculado o caminho e apresentado novamente o mapa com o caminho em coloração diferente.
Para visualizar a conetividade de um mapa é só necessário colocar a opção correta no menu.
